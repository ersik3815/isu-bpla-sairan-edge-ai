# ИСУ БПЛА — веб‑приложение

Учебный проект демонстрирует концепцию интеллектуальной системы управления (ИСУ) для беспилотного летательного аппарата (БПЛА).  Приложение иллюстрирует, как спутниковые данные (ДЗЗ), методы нейросетевого прогнозирования и алгоритмы регрессии могут быть объединены для контроля траектории и состояния аппарата в реальном времени.

## Функции

- **Карта** — просматривайте спутниковые изображения Esri или плитки OpenStreetMap. Маршрут полёта строится вокруг озера Сайран (Алматы), отмечены препятствия и буферная зона.
- **Телеметрия** — наблюдайте графики высоты (GPS/EKF), скоростей по осям, акселерометра и качества GPS. Лог телеметрии выводит имитированные сообщения 50&nbsp;Гц.
- **Нейросеть** — визуализация архитектуры LSTM+MLP, графики потерь и точности, журнал обучения. Отдельная кнопка запускает демонстрацию обучения.
- **Анализ** — сравнение эффективности различных режимов (MLP, PID, LSTM, EKF+NN, RL). Диаграмма «радар» показывает преимущества нейро‑управления над классическим PID, гистограмма NDVI — распределение классов на спутниковом снимке.
- **Управление** — переключение режимов (автопилот, ручной, RL агент), отправка команд (взлёт, зависание, посадка, миссия), настройка параметров полёта и RL, виртуальный джойстик.

## Структура проекта

- `index.html` — главная точка входа, содержит разметку всех страниц.
- `assets/css/styles.css` — стили оформления, адаптированные под тёмную тему.
- `assets/js/` — модульные скрипты:
  - `app.js` — глобальные переменные, генерация синтетических данных и общие функции.
  - `ui.js` — навигация между разделами и базовые DOM‑утилиты.
  - `map.js` — инициализация Leaflet‑карты и анимация полёта.
  - `neural.js` — визуализация нейросети и симуляция обучения.
  - `telemetry.js` — генерация и обновление телеметрических графиков.
  - `analysis.js` — построение гистограммы NDVI и радар‑диаграммы для сравнения алгоритмов.
  - `control.js` — команды управления, переключение режимов и виртуальный джойстик.
- `assets/data/` — примеры GeoJSON с маршрутом (`flight_path.geojson`) и препятствиями (`obstacles.geojson`).

## Запуск

1. Склонируйте этот репозиторий или скачайте архив.
2. Откройте `index.html` в современном браузере (Chrome, Firefox, Edge). Для корректной работы модульных скриптов лучше использовать локальный веб‑сервер, например:

   ```bash
   python -m http.server 8000
   ```

   После запуска перейдите в браузере по адресу http://localhost:8000/isu-uav-webapp/.

3. Навигируйте по разделам с помощью бокового меню. Некоторые страницы (карта, телеметрия, нейросеть) инициализируются только при первом открытии.
